# Feed the City

Volunteer coordination app for buying sandwich ingredients at scale (~200 volunteers, ~1500 sandwiches).

## Stack
Flask + SQLite + Tailwind CDN

## Dev Commands
- `pip install -r requirements.txt`
- `python3 app.py` (runs on localhost:5000)

## Conventions
- Mobile-first — volunteers use phones while shopping
- Keep it simple — this is for a volunteer event, not enterprise software
- No over-engineering — minimal dependencies, no ORM, no build complexity
